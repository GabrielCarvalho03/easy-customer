# Customers Microfrontend

![Project Logo](https://via.placeholder.com/150)

## Descrição

O projeto `customers-microfrontend` é um microfrontend desenvolvido com React e Vite. Ele permite a visualização e gerenciamento de clientes, incluindo funcionalidades de paginação e modais responsivos.

## Funcionalidades

- Visualização de clientes
- Paginação de clientes
- Modais responsivos
- Integração com API externa

## Tecnologias Utilizadas

- React
- Vite
- TypeScript
- Styled-components
- Axios
- React Hook Form
- React Router DOM
- Zustand

## Pré-requisitos

Antes de começar, certifique-se de ter o Node.js e o npm (ou yarn) instalados na sua máquina.
